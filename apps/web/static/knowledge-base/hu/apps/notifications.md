# Értesítések alkalmazás

## Rövid összefoglaló
Az Értesítések alkalmazás a Racona rendszer központi értesítési és riasztási központja. Itt kezelheted az összes rendszer- és alkalmazásértesítést, beállíthatod a preferenciáidat és megtekintheted az értesítési előzményeket.

## Alkalmazás elérése
**Elérési út:** Indító menü → Értesítések
**Ikon:** Csengő vagy értesítési ikon
**Tálca hozzáférés:** Értesítések ikon a tálcán
**Gyorsbillentyű:** Nincs

## Főbb funkciók

### Értesítési központ
- **Összes értesítés**: Központi értesítési lista
- **Kategorizálás**: Típus szerinti csoportosítás
- **Prioritás jelzés**: Sürgősségi szintek megjelenítése
- **Időrendi sorrend**: Legújabb értesítések elől

### Értesítés típusok
- **Rendszer értesítések**: Szoftver frissítések, karbantartás
- **Biztonsági riasztások**: Bejelentkezési kísérletek, gyanús tevékenység
- **Alkalmazás értesítések**: Plugin és alkalmazás specifikus üzenetek
- **Chat üzenetek**: Új beszélgetési üzenetek
- **Feladat értesítések**: Munkafolyamat és deadline emlékeztetők

### Értesítési módok
- **Böngésző értesítések**: Desktop notification API
- **Tálca jelzések**: Számláló badge és ikon változás
- **Hang jelzések**: Opcionális audio értesítések
- **Vizuális jelzések**: Villogás és színváltozás

## Értesítési felület

### Értesítési lista
- **Kártya nézet**: Részletes értesítési kártyák
- **Kompakt nézet**: Tömör lista formátum
- **Csoportosítás**: Alkalmazás vagy típus szerint
- **Szűrés**: Olvasott/olvasatlan, prioritás, dátum

### Értesítési kártya elemek
- **Feladó**: Értesítést küldő alkalmazás vagy rendszer
- **Cím**: Értesítés főcíme
- **Tartalom**: Részletes üzenet szöveg
- **Időbélyeg**: Értesítés érkezési ideje
- **Műveletek**: Gyors műveleti gombok

### Interakciós lehetőségek
- **Megjelölés olvasottként**: Egyedi vagy tömeges jelölés
- **Törlés**: Értesítések eltávolítása
- **Archíválás**: Hosszú távú tárolás
- **Válasz műveletek**: Gyors válasz opciók

## Hibaelhárítás

### Értesítések nem érkeznek meg

#### Böngésző engedélyek
- **Notification permission**: Böngésző beállítások ellenőrzése
- **Site permissions**: Oldal-specifikus engedélyek
- **Do Not Disturb**: Operációs rendszer csendes mód
- **Browser blocking**: Reklámblokkolók és biztonsági szoftverek

#### Service Worker problémák
- **Registration failed**: Service Worker regisztrációs hiba
- **Update issues**: Service Worker frissítési problémák
- **Cache problems**: Gyorsítótár ütközések
- **Script errors**: JavaScript futási hibák

### Értesítések túl gyakran érkeznek

#### Spam szűrés
- **Rate limiting**: Értesítési gyakoriság korlátozása
- **Duplicate detection**: Duplikált értesítések szűrése
- **Priority filtering**: Csak fontos értesítések engedélyezése
- **Time-based rules**: Időalapú szűrési szabályok

#### Beállítások optimalizálás
- **Notification preferences**: Értesítési preferenciák finomhangolása
- **Application settings**: Alkalmazás-specifikus beállítások
- **Quiet hours**: Csendes órák kiterjesztése
- **Batch notifications**: Csoportos értesítések engedélyezése

### Teljesítmény problémák

#### Memória használat
- **Notification cleanup**: Régi értesítések törlése
- **Cache management**: Gyorsítótár kezelés optimalizálása
- **Event listener cleanup**: Eseménykezelők megfelelő törlése
- **Memory leaks**: Memóriaszivárgások javítása

#### Hálózati teljesítmény
- **Push server load**: Értesítési szerver terhelés
- **Connection pooling**: Kapcsolat újrafelhasználás
- **Compression**: Üzenet tömörítés
- **CDN usage**: Tartalom kézbesítési hálózat használata

## Fejlesztői információk

### Értesítés küldési API
```javascript
// Értesítés küldése
await fetch('/api/notifications', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    title: 'Értesítés címe',
    message: 'Értesítés tartalma',
    type: 'info',
    priority: 'normal',
    actions: [
      { action: 'view', title: 'Megtekintés' },
      { action: 'dismiss', title: 'Elvetés' }
    ]
  })
});
```

### Service Worker regisztráció
```javascript
// Service Worker regisztráció
if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('/sw.js')
    .then(registration => {
      console.log('SW registered:', registration);
    })
    .catch(error => {
      console.log('SW registration failed:', error);
    });
}
```

### Push értesítés feliratkozás
```javascript
// Push értesítés feliratkozás
async function subscribeToPush() {
  const registration = await navigator.serviceWorker.ready;
  const subscription = await registration.pushManager.subscribe({
    userVisibleOnly: true,
    applicationServerKey: urlBase64ToUint8Array(vapidPublicKey)
  });

  // Feliratkozás küldése a szerverre
  await fetch('/api/push-subscription', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(subscription)
  });
}
```
