# Navigáció és felhasználói felület

## Rövid összefoglaló
A Racona egy teljes asztali környezetet biztosít böngészőben. Tartalmazza a tálcát, start menüt, ablakkezelést és asztali funkciókat.

## Asztal (Desktop)

### Alapvető elemek
- **Háttérkép**: Testreszabható háttér (szín, kép, videó)
- **Asztali ikonok**: Alkalmazás parancsikonok
- **Jobb klikk menü**: Gyors műveletek és beállítások
- **Drag & Drop**: Fájlok és ikonok mozgatása

### Asztali műveletek
- **Egy kattintás**: Ikon kijelölése
- **Dupla kattintás**: Alkalmazás indítása (beállítástól függően)
- **Jobb klikk**: Helyi menü megnyitása
- **Drag**: Ikon mozgatása

## Tálca (Taskbar)

### Tálca elemei
- **Start gomb**: Indítópult megnyitása (bal szélső)
- **Alkalmazás gombok**: Futó alkalmazások
- **Rendszer tálca**: Értesítések, óra, rendszer ikonok (jobb szélső)
- **Gyors indítás**: Rögzített alkalmazások

### Tálca műveletek
- **Kattintás alkalmazás gombra**: Alkalmazás előtérbe hozása
- **Jobb klikk alkalmazás gombra**: Alkalmazás menü
- **Hover**: Ablak előnézet (ha engedélyezett)
- **Drag**: Alkalmazás gombok átrendezése

## Start Menü (Indítópult)

### Start menü felépítése
- **Keresőmező**: Alkalmazások és beállítások keresése
- **Alkalmazás lista**: Kategóriák szerint rendezve
- **Gyakran használt**: Legutóbb használt alkalmazások
- **Rendszer gombok**: Kijelentkezés, újraindítás, stb.

### Start menü használata
1. **Megnyitás**: Kattints a Start gombra vagy nyomd meg a Windows billentyűt
2. **Keresés**: Kezdj el gépelni az alkalmazás nevét
3. **Böngészés**: Navigálj a kategóriák között
4. **Indítás**: Kattints az alkalmazásra

### Keresési funkciók
- **Azonnali keresés**: Gépelés közben szűrés
- **Kategória szűrés**: Csak alkalmazások, beállítások, stb.
- **Gyors hozzáférés**: Gyakran használt elemek előre sorolása

## Ablakkezelés

### Ablak állapotok
- **Normál**: Átméretezhető ablak
- **Maximalizált**: Teljes képernyős
- **Minimalizált**: Csak a tálcán látható
- **Fókuszban**: Aktív, használható ablak

### Ablak műveletek
- **Mozgatás**: Címsor drag-elése
- **Átméretezés**: Szél vagy sarok húzása
- **Maximalizás**: Dupla kattintás a címsoron vagy maximize gomb
- **Minimalizás**: Minimize gomb
- **Bezárás**: X gomb vagy Alt+F4

### Ablak pozicionálás
- **Snap to edges**: Ablak szélekhez igazítása
- **Snap to grid**: Rács alapú elhelyezés
- **Multi-monitor**: Több képernyő támogatás (ha elérhető)

## Billentyű kombinációk

### Rendszer navigáció
- **Windows/Cmd**: Start menü megnyitása
- **Alt + Tab**: Alkalmazások közötti váltás
- **Ctrl + Shift + Esc**: Feladatkezelő
- **F11**: Teljes képernyős mód
- **Alt + F4**: Aktív ablak bezárása

### Ablakkezelés
- **Windows + ↑**: Ablak maximalizálása
- **Windows + ↓**: Ablak visszaállítása/minimalizálása
- **Windows + ←/→**: Ablak bal/jobb félre igazítása
- **Alt + Space**: Ablak rendszer menü

## Értesítések

### Értesítési terület
- **Rendszer tálca**: Értesítési ikonok
- **Toast üzenetek**: Felugró értesítések
- **Értesítési központ**: Összes értesítés egy helyen

### Értesítés típusok
- **Információs**: Kék háttér, info ikon
- **Siker**: Zöld háttér, pipa ikon
- **Figyelmeztetés**: Sárga háttér, figyelmeztető ikon
- **Hiba**: Piros háttér, X ikon

## Gesztusok és érintés

### Egér gesztusok
- **Scroll**: Görgetés listákban és oldalakon
- **Right-click**: Helyi menü
- **Middle-click**: Új tab vagy speciális funkció
- **Drag & Drop**: Elemek mozgatása

### Érintőképernyő (ha támogatott)
- **Tap**: Kattintás
- **Long press**: Jobb klikk
- **Swipe**: Görgetés
- **Pinch**: Nagyítás/kicsinyítés

## Hibaelhárítás

**A tálca nem jelenik meg:**
→ Nyomd meg a Windows billentyűt
→ Ellenőrizd az automatikus elrejtés beállítást
→ Frissítsd az oldalt (F5)

**Az alkalmazások nem indulnak:**
→ Ellenőrizd a Start menüt
→ Próbáld meg a keresést
→ Ellenőrizd a jogosultságokat

**Az ablakok nem mozgathatók:**
→ Ellenőrizd, hogy nem maximalizált-e az ablak
→ Próbáld meg a címsor közepét húzni
→ Használd az Alt + Space menüt

**A billentyű kombinációk nem működnek:**
→ Ellenőrizd, hogy a böngésző fókuszban van-e
→ Próbáld meg újratölteni az oldalt
→ Ellenőrizd a böngésző beállításokat

**Lassú navigáció:**
→ Zárd be a nem használt alkalmazásokat
→ Ellenőrizd az internet kapcsolatot
→ Próbáld meg újraindítani a böngészőt
