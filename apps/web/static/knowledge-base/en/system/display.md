# Display and Background

## Quick Summary
The Racona system offers complete customization of appearance. You can configure themes, color schemes, backgrounds, desktop and taskbar appearance.

## Theme and Appearance Settings
**Path:** Settings → Appearance

1. Open the **Settings** application
2. Select the **Appearance** menu item
3. Configurable options:
   - **Theme Presets**: Pre-configured themes
   - **Theme Mode**: Light, dark or automatic
   - **Color Scheme**: Custom color selection
   - **Font Size**: Text size adjustment

### Theme Presets
- Choose from pre-configured themes
- Each theme includes: color scheme, theme mode and background
- **Quick Apply**: Complete theme setup with one click

### Theme Mode
- **Light**: Light color scheme
- **Dark**: Dark color scheme
- **Automatic**: System setting or time-based

### Color Scheme Customization
- Choose from predefined color schemes
- Custom color setup with color picker
- Instant preview of changes

## Background Settings
**Path:** Settings → Desktop → Background

1. Open the **Settings** application
2. Select the **Desktop** menu item
3. Click the **Background** option
4. Choose background type:
   - **Color**: Solid color background
   - **Image**: Use background image
   - **Video**: Use video background

### Image Background Setup
1. Select the **Image** option
2. Choose source:
   - **System Images**: Pre-installed backgrounds
   - **Upload Your Own Image**: Upload custom image
3. Select image from thumbnail preview
4. Optional effects:
   - **Blur**: 0-30 pixel blur (with slider)
   - **Grayscale**: Black and white effect
5. Click the **Apply** button

**Supported formats:** JPG, JPEG, PNG, WebP
**Maximum file size:** 10 MB

### Video Background Setup
1. Select the **Video** option
2. Upload video or choose from system videos
3. **Supported formats:** MP4, WebM, OGG
4. **Warning:** Video background is resource-intensive, disable in Performance settings on slower devices

## Desktop Settings
**Path:** Settings → Desktop → General

### Desktop Shortcuts
- **Opening Mode**: Single click or double click
- **Icon Arrangement**: Automatic grid or free placement
- **Icon Size**: Small, medium, large

### Right-click Menu
- **Customize Background**: Quick access to background settings
- **Arrange Icons**: Automatic grid arrangement
- **Hide/Show Icons**: Manage all desktop icons

## Taskbar Settings
**Path:** Settings → Desktop → Taskbar

### Position and Appearance
- **Position**: Bottom, top, left, right
- **Auto-hide**: Hide taskbar when inactive
- **Transparency**: Taskbar transparency setting

### Taskbar Elements
- **Show Clock**: Enable/disable
- **Application Buttons**: Show running applications
- **System Tray**: Notifications and system icons
- **Start Button**: Show launcher button

## Start Menu Settings
**Path:** Settings → Desktop → Start Panel

### View Settings
- **Display Mode**: Grid or list view
- **Application Grouping**: Sort by categories
- **Search**: Show search field

### Search Settings
- **Instant Search**: Search while typing
- **Search Categories**: Applications, files, settings

## Troubleshooting

**Background doesn't change:**
→ Check file format (JPG, PNG, WebP)
→ Try restarting the system
→ Check file size (max 10 MB)

**Theme doesn't switch:**
→ Refresh the page (F5)
→ Check if not in automatic mode
→ Try another theme first

**Video background doesn't play:**
→ Check video format (MP4, WebM, OGG)
→ Disable video background in Performance settings
→ Try with image first

**Taskbar doesn't appear:**
→ Check auto-hide setting
→ Try different position
→ Restart the system

**Desktop icons disappeared:**
→ Right-click desktop → Show Icons
→ Check Desktop settings
→ Try recreating shortcuts
