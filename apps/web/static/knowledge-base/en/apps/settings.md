# Settings Application

## Quick Summary
The Settings application is the central configuration hub of the Racona system. Here you can manage your account, security settings, appearance, desktop environment, performance, and language settings.

## Application Access
**Access Path:** Start Menu → Settings
**Icon:** Gear wheel
**Keyboard Shortcut:** None

## Main Menu Structure

### 1. Account
**Access Path:** Settings → Account

#### Profile Settings
- **Profile Picture Management**: Upload custom image or use default
- **Personal Information**: Edit full name, username, email address
- **Account Type**: Display Google account or email account
- **Groups and Roles**: View assigned groups and roles
- **Registration Date**: Account creation timestamp

#### Validation Rules
- **Name**: Required field
- **Username**: 3-50 characters, letters, numbers and underscores only
- **Email**: Read-only (synced from external provider)

### 2. Security
**Access Path:** Settings → Security

#### Password Change
- **Current Password**: Confirmation required
- **New Password**: Minimum 8 characters
- **Password Confirmation**: Match verification
- **Security Requirements**: Strong, unique password recommended

#### Two-Factor Authentication (2FA)
- **Enable/Disable**: Password confirmation required
- **QR Code**: Scan with authenticator app
- **Manual Setup**: Enter secret key manually
- **Verification Code**: 6-digit TOTP code
- **Backup Codes**: Single-use recovery codes
- **Backup Code Management**: Regenerate, copy, download

### 3. Appearance
**Access Path:** Settings → Appearance

#### Theme Settings
- **Desktop Theme Mode**: Light, dark, or automatic
- **Taskbar Theme Mode**: Independent theme for taskbar
- **Theme Modes**:
  - Light Mode: Light background, dark text
  - Dark Mode: Dark background, light text
  - Auto Mode: Follow system settings

#### Colors
- **Primary Color**: Application accent color
- **Predefined Colors**: Quick selection
- **Custom Color**: Hue slider adjustment
- **Application Areas**: Buttons, links, interactive elements

#### Font Size
- **System Font Size**: Small, medium, large
- **Effect**: All system text sizes
- **Recommendation**: Medium size for optimal readability

#### Themes (Presets)
- **Predefined Themes**: One-click application
- **Theme Components**: Theme mode, color, font size, background
- **Custom Settings**: Return to individual configuration

### 4. Desktop
**Access Path:** Settings → Desktop

#### 4.1 General
**Access Path:** Settings → Desktop → General

##### Open Shortcuts
- **Single Click**: Faster access
- **Double Click**: Prevent accidental opening
- **Recommendation**: Based on user preference

#### 4.2 Background
**Access Path:** Settings → Desktop → Background

##### Background Types
- **Color**: Solid color background
- **Image**: Static wallpaper
- **Video**: Animated background (muted, optimized)

##### Background Color Settings
- **Predefined Colors**: Quick selection
- **Color Picker**: Create custom color
- **Instant Application**: Real-time preview

##### Background Image Settings
- **Predefined Images**: Built-in wallpapers
- **Upload Custom Image**: Personal wallpaper
- **Supported Formats**: JPG, PNG, WebP
- **Auto Scaling**: Adjust to screen size
- **Image Management**: Upload, select, delete

##### Background Video Settings
- **Predefined Videos**: Built-in animations
- **Continuous Loop**: Infinite repeat
- **Muted Playback**: No audio
- **Performance Optimization**: Minimal system load

##### Visual Effects
- **Blur**: Adjustable intensity
- **Grayscale**: Black and white display
- **Combinable Effects**: Blur + grayscale together

#### 4.3 Taskbar
**Access Path:** Settings → Desktop → Taskbar

##### Position Settings
- **Top**: At screen top
- **Bottom**: At screen bottom (default)
- **Navigation Benefits**: More comfortable access

##### Style Settings
- **Classic**: Full screen width
- **Floating**: With margins and rounded corners
- **Visual Differences**: Appearance only, same functionality

##### Taskbar Items
- **Clock**: Display current time
- **Theme Switcher**: Light/dark theme toggle
- **App Launcher**: GUID-based application launch
- **Messages**: Quick access to chat messages
- **Notifications**: System notifications
- **Item Management**: Enable/disable individually

#### 4.4 Start Panel
**Access Path:** Settings → Desktop → Start Panel

##### Display Modes
- **Grid View**: Grid layout with icons
- **List View**: Detailed information per row
- **Selection Criteria**: Personal preference and usage habits

### 5. Performance
**Access Path:** Settings → Performance

#### Performance Optimization
- **Faster Operation**: Reduce visual effects
- **Window Dragging**: Hide content during movement
- **Window Preview**: Disable on slower devices
- **Target Audience**: Devices with weaker hardware

#### Window Preview
- **Function**: Preview images on taskbar
- **Operation**: Hover effect on application icons
- **Benefits**: Easier navigation between open windows
- **Performance**: Memory and CPU requirements

#### Preview Image Size
- **Small**: Minimal resource requirements
- **Medium**: Recommended setting
- **Large**: More detailed preview
- **Huge**: Maximum detail, high resource usage

### 6. Language & Region
**Access Path:** Settings → Language & Region
**Condition**: Only appears in multilingual systems

#### Language Selection
- **Available Languages**: Languages installed in system
- **Immediate Effect**: Takes effect immediately after change
- **Full System**: All text affected
- **User Interface**: Menus, buttons, messages

### 7. About
**Access Path:** Settings → About

#### System Information
- **Version**: Current Racona version
- **Description**: "Modern web desktop environment for efficient work"
- **Version History**: Changelog access
- **Copyright**: © 2025 Racona

## Common Operations

### Saving Settings
1. Modify the desired setting
2. System automatically saves changes
3. Confirmation message appears on successful save

### Resetting Settings
- "Reset" button available for some settings
- Return to default values
- Confirmation required before reset

### Troubleshooting

#### Settings Not Saving
- **Check**: Proper permissions
- **Try**: Page refresh and retry
- **On Error**: Notify system administrator

#### Theme Not Changing
- **Wait**: Few seconds for change to take effect
- **Clear Cache**: Empty browser cache
- **Reload**: Refresh page

#### 2FA Issues
- **Time Sync**: Check device clock synchronization
- **Backup Codes**: Use recovery codes
- **Re-setup**: Disable and re-enable 2FA

#### Background Image Not Showing
- **File Format**: Check supported format (JPG, PNG, WebP)
- **File Size**: Large files may cause issues
- **Browser Cache**: Clear cache and retry

## Security Notes

### Password Security
- Use strong, unique passwords
- Don't reuse passwords from other services
- Change passwords regularly

### 2FA Recommendations
- Always enable two-factor authentication
- Save backup codes in a secure location
- Use trusted authenticator apps (Google Authenticator, Authy)

### Profile Data
- Only provide accurate information
- Regularly review profile information
- Report suspicious activities
